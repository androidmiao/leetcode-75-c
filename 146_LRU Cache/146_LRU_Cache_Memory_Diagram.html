146. LRU Cache — 記憶體位址 / 陣列索引 詳細說明圖

使用 Example 1：capacity = 2，追蹤每一步的 Hash Table 陣列 + 雙向鏈結串列的完整記憶體狀態

假設的記憶體位址（方便追蹤指標）
LRUCache 結構體: 0xA000  |  Head 哨兵: 0xB000  |  Tail 哨兵: 0xB020
資料節點依序分配: 0xC000, 0xC020, 0xC040, 0xC060
HashNode 依序分配: 0xD000, 0xD020, 0xD040, 0xD060

Hash 函式: hash(key) = key % 10007 → 在此例中 hash(1)=1, hash(2)=2, hash(3)=3, hash(4)=4

Example 1 操作序列

操作: LRUCache(2), put(1,1), put(2,2), get(1), put(3,3), get(2), put(4,4), get(1), get(3), get(4) 回傳: null, null, null, 1, null, -1, null, -1, 3, 4
0
LRUCache(2) — 初始化，capacity = 2

malloc LRUCache、Head 哨兵、Tail 哨兵；hash table 全部清零

2
put(2, 2) — 插入 key=2, value=2

hashFind(2) = NULL → 新建 Node 0xC020 + HashNode 0xD020，insertBeforeTail，size 1→2（已滿）

新分配的記憶體： Node @ 0xC020 { key=2, val=2, prev=?, next=? } HashNode @ 0xD020 { key=2, node=0xC020, next=NULL } 雙向鏈結串列 size = 2（已達 capacity） 0xB000 0xC000 0xC020 0xB020 ┌──────────────┐ ┌──────────────┐ ┌──────────────┐ ┌──────────────┐ │ HEAD 哨兵 │ │ key=1 val=1 │ │ key=2 val=2 │ │ TAIL 哨兵 │ │ prev = NULL │──→│ prev = 0xB000│──→│ prev = 0xC000│──→│ prev = 0xC020│ │ next = 0xC000│←──│ next = 0xC020│←──│ next = 0xB020│←──│ next = NULL │ └──────────────┘ └──────────────┘ └──────────────┘ └──────────────┘ HEAD ──→ [1,1] ──→ [2,2] ──→ TAIL 最舊 ←─────────────────────→ 最新 Hash Table ┌───────┬───────────────────────────────────────────────────────┐ │ index │ bucket → HashNode chain │ ├───────┼───────────────────────────────────────────────────────┤ │ 0 │ NULL │ │ 1 │ 0xD000 ─→ { key=1, node=0xC000, next=NULL } │ │ 2 │ 0xD020 ─→ { key=2, node=0xC020, next=NULL } │ │ 3 │ NULL │ │ 4 │ NULL │ └───────┴───────────────────────────────────────────────────────┘
3
get(1) → 回傳 1，將 [1,1] 移至最近端

hashFind(1) = 0xC000 → removeNode(0xC000) → insertBeforeTail(0xC000) → return 1

Step 3a: removeNode(0xC000) — 從鏈結串列中摘除 [1,1] 修改指標： 0xB000(HEAD).next = 0xC020 （原本 = 0xC000） 0xC020([2,2]).prev = 0xB000 （原本 = 0xC000） HEAD ──→ [2,2] ──→ TAIL [1,1] 暫時孤立 Step 3b: insertBeforeTail(0xC000) — 將 [1,1] 插入 TAIL 之前 修改指標： 0xC000([1,1]).prev = 0xC020 （[2,2] 的位址） 0xC000([1,1]).next = 0xB020 （TAIL 的位址） 0xC020([2,2]).next = 0xC000 （原本 = 0xB020） 0xB020(TAIL).prev = 0xC000 （原本 = 0xC020） 雙向鏈結串列 size = 2（不變） 0xB000 0xC020 0xC000 0xB020 ┌──────────────┐ ┌──────────────┐ ┌──────────────┐ ┌──────────────┐ │ HEAD 哨兵 │ │ key=2 val=2 │ │ key=1 val=1 │ │ TAIL 哨兵 │ │ prev = NULL │──→│ prev = 0xB000│──→│ prev = 0xC020│──→│ prev = 0xC000│ │ next = 0xC020│←──│ next = 0xC000│←──│ next = 0xB020│←──│ next = NULL │ └──────────────┘ └──────────────┘ └──────────────┘ └──────────────┘ HEAD ──→ [2,2] ──→ [1,1] ──→ TAIL 最舊 最新(剛存取) Hash Table（不變，只是鏈結串列順序改了，HashNode 指向的 Node* 沒變） ┌───────┬───────────────────────────────────────────────────────┐ │ 1 │ 0xD000 ─→ { key=1, node=0xC000, next=NULL } │ │ 2 │ 0xD020 ─→ { key=2, node=0xC020, next=NULL } │ └───────┴───────────────────────────────────────────────────────┘
4
put(3, 3) — 插入 key=3，淘汰 LRU key=2

hashFind(3) = NULL → 新建 Node 0xC040 + HashNode 0xD040 → size 2→3 > capacity → 淘汰 head->next = 0xC020 (key=2)

Step 4a: 新建節點並插入 Node @ 0xC040 { key=3, val=3 } HashNode @ 0xD040 { key=3, node=0xC040, next=NULL } → insertBeforeTail(0xC040) → size = 3（超過 capacity=2！） Step 4b: 淘汰最舊節點 lru = head->next = 0xC020 (key=2) removeNode(0xC020) ← 從鏈結串列移除 hashRemove(obj, 2) ← 從 hash table[2] 移除 HashNode 0xD020 free(0xC020) ← 釋放 Node 記憶體 size = 3 → 2 雙向鏈結串列 size = 2 0xB000 0xC000 0xC040 0xB020 ┌──────────────┐ ┌──────────────┐ ┌──────────────┐ ┌──────────────┐ │ HEAD 哨兵 │ │ key=1 val=1 │ │ key=3 val=3 │ │ TAIL 哨兵 │ │ prev = NULL │──→│ prev = 0xB000│──→│ prev = 0xC000│──→│ prev = 0xC040│ │ next = 0xC000│←──│ next = 0xC040│←──│ next = 0xB020│←──│ next = NULL │ └──────────────┘ └──────────────┘ └──────────────┘ └──────────────┘ HEAD ──→ [1,1] ──→ [3,3] ──→ TAIL 最舊 最新 已釋放: Node 0xC020 (key=2) ✗ 已釋放: HashNode 0xD020 (key=2) ✗ Hash Table ┌───────┬───────────────────────────────────────────────────────┐ │ 1 │ 0xD000 ─→ { key=1, node=0xC000, next=NULL } │ │ 2 │ NULL ← 已移除 │ │ 3 │ 0xD040 ─→ { key=3, node=0xC040, next=NULL } │ └───────┴───────────────────────────────────────────────────────┘
5
get(2) → 回傳 -1（已被淘汰）

hashFind(2) → table[2] = NULL → return -1。鏈結串列和 hash table 完全不變。

查找過程： h = hash(2) = 2 table[2] = NULL ← key=2 在 Step 4 已被淘汰 → return -1 雙向鏈結串列（不變） HEAD ──→ [1,1] ──→ [3,3] ──→ TAIL
6
put(4, 4) — 插入 key=4，淘汰 LRU key=1

hashFind(4) = NULL → 新建 Node 0xC060 + HashNode 0xD060 → size 2→3 > capacity → 淘汰 head->next = 0xC000 (key=1)

Step 6a: 新建節點並插入 Node @ 0xC060 { key=4, val=4 } HashNode @ 0xD060 { key=4, node=0xC060, next=NULL } → insertBeforeTail(0xC060) → size = 3（超過 capacity=2！） Step 6b: 淘汰最舊節點 lru = head->next = 0xC000 (key=1) removeNode(0xC000) ← 從鏈結串列移除 hashRemove(obj, 1) ← 從 hash table[1] 移除 HashNode 0xD000 free(0xC000) ← 釋放 Node 記憶體 size = 3 → 2 雙向鏈結串列 size = 2 0xB000 0xC040 0xC060 0xB020 ┌──────────────┐ ┌──────────────┐ ┌──────────────┐ ┌──────────────┐ │ HEAD 哨兵 │ │ key=3 val=3 │ │ key=4 val=4 │ │ TAIL 哨兵 │ │ prev = NULL │──→│ prev = 0xB000│──→│ prev = 0xC040│──→│ prev = 0xC060│ │ next = 0xC040│←──│ next = 0xC060│←──│ next = 0xB020│←──│ next = NULL │ └──────────────┘ └──────────────┘ └──────────────┘ └──────────────┘ HEAD ──→ [3,3] ──→ [4,4] ──→ TAIL 最舊 最新 已釋放: Node 0xC000 (key=1) ✗ 已釋放: HashNode 0xD000 (key=1) ✗ Hash Table ┌───────┬───────────────────────────────────────────────────────┐ │ 1 │ NULL ← 已移除 │ │ 2 │ NULL ← 早已移除 │ │ 3 │ 0xD040 ─→ { key=3, node=0xC040, next=NULL } │ │ 4 │ 0xD060 ─→ { key=4, node=0xC060, next=NULL } │ └───────┴───────────────────────────────────────────────────────┘
7
get(1) → 回傳 -1（已被淘汰）
