Given the `head` of a singly linked list, return `true` if it is a palindrome or `false` otherwise.

**Example 1:**

Input: `head = [1,2,2,1]`
Output: `true`

**Example 2:**

Input: `head = [1,2]`
Output: `false`

**Constraints:**

- The number of nodes in the list is in the range `[1, 10^5]`.
- `0 <= Node.val <= 9`

**Follow up:** Could you do it in `O(n)` time and `O(1)` space?

---

給定一個 singly linked list 的頭節點 `head`，如果它是回文（palindrome）則回傳 `true`，否則回傳 `false`。

**範例 1：**

輸入：`head = [1,2,2,1]`
輸出：`true`

**範例 2：**

輸入：`head = [1,2]`
輸出：`false`

**限制條件：**

- linked list 中節點的數量範圍為 `[1, 10^5]`。
- `0 <= Node.val <= 9`

**進階問題：** 你能否在 `O(n)` 時間與 `O(1)` 額外空間內解決這道題？
