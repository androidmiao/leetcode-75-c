## 3. 搜尋/操作過程追蹤

### 算法概述

```
邏輯流程:
  1. 初始化 max_prod, min_prod, result 為 nums[0]
  2. 從 i=1 開始迴圈到末尾
  3. 在每一步計算:
     - 三個候選值: nums[i], nums[i]*max_prod, nums[i]*min_prod
     - temp_max = 保存當前最大值
     - min_prod = min(nums[i], nums[i]*temp_max, nums[i]*min_prod)
     - max_prod = max(nums[i], nums[i]*temp_max, nums[i]*min_prod)
     - result = max(result, max_prod)
```

### 範例 1 完整追蹤: nums = [2, 3, -2, 4]

```
初始化 (i=-1 概念上):
─────────────────────────────────────
  max_prod = nums[0] = 2
  min_prod = nums[0] = 2
  result   = nums[0] = 2

指標位置:
  ┌──────────┬──────────┬──────────┬──────────┐
  │ [2]      │ [3]      │ [-2]     │ [4]      │
  │ 0xA000   │ 0xA004   │ 0xA008   │ 0xA00C   │
  └──────────┴──────────┴──────────┴──────────┘
  ▲
  (i=-1, 初始化完成)


步驟 1 (i=1):
─────────────────────────────────────────────────────────────
  訪問位址: 0xA004 (nums[1] = 3)

  計算候選值:
    • curr = nums[1] = 3
    • 候選1: curr = 3
    • 候選2: curr × max_prod = 3 × 2 = 6
    • 候選3: curr × min_prod = 3 × 2 = 6

  更新:
    • temp_max = max(3, 6, 6) = 6  ◀── 保存舊的 max_prod
    • min_prod = min(3, 6, 6) = 3
    • max_prod = max(3, 6, 6) = 6
    • result = max(2, 6) = 6

  指標位置:
  ┌──────────┬──────────┬──────────┬──────────┐
  │ [2]      │ [3]      │ [-2]     │ [4]      │
  │ 0xA000   │ 0xA004   │ 0xA008   │ 0xA00C   │
  └──────────┴──────────┴──────────┴──────────┘
           ▲
         (i=1)


步驟 2 (i=2):
─────────────────────────────────────────────────────────────
  訪問位址: 0xA008 (nums[2] = -2)

  計算候選值:
    • curr = nums[2] = -2
    • 候選1: curr = -2
    • 候選2: curr × max_prod = -2 × 6 = -12
    • 候選3: curr × min_prod = -2 × 3 = -6

  更新:
    • temp_max = max(-2, -12, -6) = -2  ◀── 此時 max_prod=6
    • min_prod = min(-2, -12, -6) = -12
    • max_prod = max(-2, -12, -6) = -2
    • result = max(6, -2) = 6

  指標位置:
  ┌──────────┬──────────┬──────────┬──────────┐
  │ [2]      │ [3]      │ [-2]     │ [4]      │
  │ 0xA000   │ 0xA004   │ 0xA008   │ 0xA00C   │
  └──────────┴──────────┴──────────┴──────────┘
                      ▲
                    (i=2)


步驟 3 (i=3):
─────────────────────────────────────────────────────────────
  訪問位址: 0xA00C (nums[3] = 4)

  計算候選值:
    • curr = nums[3] = 4
    • 候選1: curr = 4
    • 候選2: curr × max_prod = 4 × (-2) = -8
    • 候選3: curr × min_prod = 4 × (-12) = -48

  更新:
    • temp_max = max(4, -8, -48) = 4  ◀── 此時 max_prod=-2
    • min_prod = min(4, -8, -48) = -48
    • max_prod = max(4, -8, -48) = 4
    • result = max(6, 4) = 6

  指標位置:
  ┌──────────┬──────────┬──────────┬──────────┐
  │ [2]      │ [3]      │ [-2]     │ [4]      │
  │ 0xA000   │ 0xA004   │ 0xA008   │ 0xA00C   │
  └──────────┴──────────┴──────────┴──────────┘
                              ▲
                            (i=3)

  迴圈結束 (i=4 超出範圍)


最終結果: result = 6  ✓
子陣列: [2, 3] 或 [-2, 4] (乘積=6)
```

## 5. 記憶體存取模式分析

### 連續陣列存取 (當前實作)

```
存取模式 (Stride-1, 順序存取):
─────────────────────────────────────────

位址線性序列:
  0xA000 ──▶ 0xA004 ──▶ 0xA008 ──▶ 0xA00C ──▶ (結束)
    ↓         ↓         ↓         ↓
  [2]       [3]       [-2]      [4]
  讀 ✓      讀 ✓      讀 ✓      讀 ✓

特性:
  • Stride = 4 位元組 (恆定)
  • 模式: 完全順序，無分支
  • 預測性: 100% (可預測跳躍)

L1 快取行分析 (64 位元組):
─────────────────────────────────────────────────────────

快取線 1 (0xA000-0xA03F):
┌────┬────┬────┬────┬────┬────┬────┬────────┐
│[2] │[3] │[-2]│[4] │    │    │    │(空)    │
└────┴────┴────┴────┴────┴────┴────┴────────┘
4B   4B   4B   4B   16B    (unused)

✓ 命中率: 4/4 = 100% (全部 4 個 int 在同一快取線)
✓ 利用率: 16B / 64B = 25% (快取線的 1/4)
✓ 預取友善性: 優秀 (順序存取可自動預取)

位址計算成本:
  • 每次反覆: 1 × 讀 (load)
  • 位址算式: base_addr + i*4
  • CPU 周期: 1-2 (快取命中)
```

## 7. 概念總結

### 記憶體特性

| 維度 | 描述 |
|------|------|
| **空間複雜度** | O(1) - 僅 5 個標量變數 (max_prod, min_prod, result, temp_max, curr) |
| **時間複雜度** | O(n) - 單次迴圈遍歷陣列 |
| **快取友善性** | ✓ 優秀 - Stride-1 順序存取，100% L1 命中率 |
| **預取效率** | ✓ 優秀 - 硬體預取器可預測 |
| **分支預測** | ✓ 優秀 - 單個迴圈，邊界分支可預測 |
| **TLB 效率** | ✓ 優秀 - 陣列可能在單一頁面內 |
| **記憶體寶藏空間** | 低 - 不產生臨時物件，無動態分配 |

## 附錄: 完整程式碼參考

```c
#include <stdio.h>
#include <string.h>
#include <limits.h>

int maxProduct(int* nums, int numsSize) {
    if (numsSize == 0) return 0;

    // 初始化 (存取 nums[0] 在 0xA000)
    int max_prod = nums[0];   // 0x7FFF0014
    int min_prod = nums[0];   // 0x7FFF0018
    int result = nums[0];     // 0x7FFF001C

    // 迴圈遍歷 i=1 到 numsSize-1
    for (int i = 1; i < numsSize; i++) {
        // 訪問 nums[i] (位址: 0xA000 + i*4)
        int curr = nums[i];           // 0x7FFF0020
        int temp_max = max_prod;      // 0x7FFF0024 (保存)

        // 計算三個候選值並選出最小/最大
        max_prod = (curr > curr * temp_max && curr > curr * min_prod)
                    ? curr
                    : ((curr * temp_max > curr * min_prod)
                        ? curr * temp_max
                        : curr * min_prod);

        min_prod = (curr < curr * temp_max && curr < curr * min_prod)
                    ? curr
                    : ((curr * temp_max < curr * min_prod)
                        ? curr * temp_max
                        : curr * min_prod);

        // 更新全域最大值
        result = (max_prod > result) ? max_prod : result;
    }

    return result;
}

// 測試用例
int main() {
    // 範例 1
    int nums1[] = {2, 3, -2, 4};
    printf("Example 1: %d (expected 6)\n", maxProduct(nums1, 4));

    // 範例 2
    int nums2[] = {-2, 0, -1};
    printf("Example 2: %d (expected 0)\n", maxProduct(nums2, 3));

    return 0;
}
```
