# 345. Reverse Vowels of a String

**Difficulty:** Easy

## Problem Statement

Given a string `s`, reverse only all the vowels in the string and return it.

The vowels are `'a'`, `'e'`, `'i'`, `'o'`, and `'u'`, and they can appear in both lower and upper cases, more than once.

## Examples

### Example 1

- **Input:** `s = "IceCreAm"`
- **Output:** `"AcesCreIm"`
- **Explanation:** The vowels in `s` are `['I', 'e', 'e', 'A']`. On reversing the vowels, `s` becomes `"AceCreIm"`.

### Example 2

- **Input:** `s = "leetcode"`
- **Output:** `"leotcede"`

## Constraints

- `1 <= s.length <= 3 * 10^5`
- `s` consist of **printable ASCII** characters.

# 345. 反轉字串中的母音字母

**難度：** 簡單

## 題目描述

給定一個字串 `s`，只反轉字串中的所有母音字母，然後回傳結果。

母音字母為 `'a'`、`'e'`、`'i'`、`'o'` 和 `'u'`，它們可以以大寫和小寫的形式出現，且可以出現多次。

## 範例

### 範例 1

- **輸入：** `s = "IceCreAm"`
- **輸出：** `"AceCreIm"`
- **說明：** `s` 中的母音為 `['I', 'e', 'e', 'A']`。反轉母音後，`s` 變成 `"AceCreIm"`。

### 範例 2

- **輸入：** `s = "leetcode"`
- **輸出：** `"leotcede"`

## 限制條件

- `1 <= s.length <= 3 * 10^5`
- `s` 由**可列印的 ASCII** 字元組成。
